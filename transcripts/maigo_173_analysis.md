# 迷子集会 #173 分析 — 新規確認事項

## 基本情報

- 出演: 羊宮妃那（高松燈役）+ 長崎そよ

## 新たに確認された発話パターン

### 1. 「かわいい」連打（感情爆発）
[211.4s-213.1s] **「かわいい、かわいい、かわいい、かわいい、かわいい、かわいい、かわいい、かわいい」**
→ 妹の卒業エピソードに共感して9連発。テンションが上がると抑制が外れる。

### 2. 「あ、やーよー」（驚き・不満の自然な表現）
[54.1s-55.3s] 「あ、やーよー」
→ もう4月になったことへの驚き。非常にくだけた自然な言い回し。

### 3. 「なんか」超高頻度
「なんか、今思い返しと…」「なんか、基本バトミントンとか」「なんかいろいろあるよね」
→ #174 以上に頻出。思考を整理しながら話す時のフィラーとして定着。

### 4. 「そうそうそうそう」（急速同意）
[368.4s-370.9s] 「そうそうそうそう、えっとー?」
→ 相手の話に強く同意する時のパターン。

### 5. 剣道部・茶道部エピソード
[310.1s-315.0s] 「えっと、中学で剣道部と高校で、茶道部ですけど」
→ 本人的な体験談。Wikipedia の「剣道初段」が実体験として語られた。

### 6. 幼少期の遊び
[614.8s-617.8s] 「子供の頃はよく水溜まりなんかで遊んでて」
→ 素朴な子供時代の回想。飾らない語り口。

### 7. 共感の「わかる」
#174 では「うん」中心だったが、#173 ではより感情的な相槌も。

## #174 との比較

| 特徴 | #174 | #173 |
|------|------|------|
| フィラー | うん、うーん、えっと | **なんか**が追加で頻出 |
| 感情表現 | 抑制気味 | **かわいい連打**など解放的 |
| 話題 | 日常・マイブーム | **自分の過去**（部活、幼少期） |
| 同意 | うん、確かに | そうそうそうそう |
| くだけた表現 | 少ない | 「あ、やーよー」出現 |

## 結論

#173 では #174 より**リラックスした素の羊宮妃那**が出ている。
- 感情の爆発（かわいい連打）
- 幼少期の回想（水溜まり遊び）
- 自分の部活経験
- 「なんか」の多用が顕著

→ 親しい相手とのラジオでは、より**自然体でテンションの振れ幅が大きい**
