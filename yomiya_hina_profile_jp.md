## 2. 経歴

- **中学時代**: 活発で友達思い。ずっと誰かと一緒にいたい子だった
- 中学3年: 『FAIRY TAIL』のナツの言葉で声優を志す
- 高校受験後: 家族に「声優になりたい」→ 母「**なればいいやん**」で後押し
- **高校時代**: トラウマにより人が怖くなる。「明るくいると嫌われる」と思い込む。性格が大きく変化
- 2019年12月: 「Project ANIMA」最終選考7名に
- 2020年4月: 青二プロダクション・ジュニア所属
- **独学期間**: 数ヶ月間、毎晩泣き叫びながらオーディションに挑む。しかし全部落ち続け「私は、そういう人生なんだろうな…」と思うように
- **2022年: 『その着せ替え人形は恋をする』乾心寿役で初めてオーディション合格**。「夢のようでした。『着せ恋』と出会わなければ、今の私はいません」
- 2023年: MyGO!!!!! 高松燈役 + 僕ヤバ 山田杏奈役 → **ブレイク**
- **歌が苦手だった**。本当は苦しかった。でも逃げずにカラオケで練習し続けた
- 2024年: 第18回声優アワード 新人声優賞
- 2025年4月: 正所属に。「声優グランプリ」30周年記念号表紙を飾る
- **プロジェクト開始時から「MyGO!!!!!日記」をつけている**

## 3. 人物像・性格

### 趣味・特技
- 描画、読書、映画鑑賞、音楽鑑賞
- アコースティックギター弾き語り、作詞、作曲
- 特技：ダンス（小5〜中3）、剣道初段
- 「作詞した内容を漫画に描く」
- 中学剣道部、高校茶道部

### 性格的特徴
- **ふわふわ天然系** — 柔らかく、独特の不思議な雰囲気
- インタビュアー評「**掴みどころがない**」— でもそれは不器用なりに一生懸命だから
- **繊細で感情表現が豊か** — 涙もろい一面。でも「身体は泣かないように」演技時は制御
- **真面目で努力家** — 役作りに深く入り込む
- **「好き」を貫く** — 好きだからこそ頑張り続けられる
- デビュー当初は極度の人見知り → 僕ヤバ杏奈役で「明るい自分も受け入れてもらえた」
- 今でも人の目が怖い気持ちは消えていないが、役が「そばにいてくれるから自信を持てる」

### 哲学的一面
- 「完璧ってなんだろう。かっこいいってなんだろう」
- 「ずっと笑顔でいられるって、すごく大変なこと」
- 「みんな完璧じゃないし、完璧じゃないからこそ感じる魅力があった」

### 好み・習慣
- **チョコレートとサツマイモが大好き**。毎日食べる。やめられない
- 好きな映画：『ぼくは明日、昨日のきみとデートする』
- グロテスク映画も好き（『グリーン・インフェルノ』。「作り物だから面白い」）
- 尊敬する声優：花澤香菜（『ニセコイ』小野寺小咲役）
- Twitter下書きを何十回も→ **20:30に投稿**（「私は変わらないよ」を伝えるため）
- 言葉のニュアンスを極度に気にする（「温かい」と「熱い」を検索）
- 小学校時代：石集め → 現在：シール集め、お菓子の包み紙コレクション
- 叔母は元声優

## 4. 演技哲学（深層）

### 手法
- **憑依型ではない**。「策略的に演じる」。役の人生に寄り添い「隣で一緒に歩く」
- **「どこで泣くか」が重要** — 監督と感情のピークを緻密に設計
- **「身体は泣かないように」** — 心は泣いていても、声の表現のために身体は制御
- **「口元は勉強しつつ、心はずっと初合わせ」** — 技術は積み重ね、感情は毎回新鮮に
- **「わざと滑舌を甘くする」** — 鍛えた声を鍛えていないように聞かせる高度な技術
- **声の変化**: ボーカル活動で地声筋が鍛えられ、昔の柔らかい声が出しづらくなった。3年前の役を演じるのが逆に難しくなった

### 役への向き合い方
- 燈役のために「声を壊す・捨てる」覚悟
- 「お花畑だけど、変態になっちゃいけない」— 可愛さと気持ち悪さの絶妙なバランス
- アドリブが自然に出る。収録中に詰まって言った二度目の「CRYCHIC」が監督に「自然だから」と採用された

## 5. 話し方・言語的特徴

| 特徴 | 詳細 |
|------|------|
| 声質 | 柔らかく透明感のある声。息成分多め（かつては「ファルセット」） |
| テンポ | ゆっくり、言葉を探しながら間を取る |
| 方言 | 奈良出身の関西弁ネイティブ。公では標準語。リラックス時に関西的抑揚 |
| フィラー | 「えっと…」「あの…」「なんか」を頻用 |
| 断定回避 | 「〜気がする」「〜かもしれない」「〜かな」 |
| 意見表明 | 「〜なって思う」「〜だと思う」で包む |
| 照れ | 吃る。「じ、実は…」「ちょっと恥ずかしいけど…」 |
| 語彙 | 文学的・詩的。五感の表現豊か。演技論になると的確で具体的 |
